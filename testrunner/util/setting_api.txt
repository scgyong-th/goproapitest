Settings
GoPro cameras have hundreds of setting options to choose from. Since these options have a complex tree of dependencies on camera state, current Preset, etc, there is no mechanism to set a desired setting option from any camera state.

To find the currently available options for a given setting, use the capability querying operations:

Get Setting Capabilities

Register for Setting Capability Updates

Operations
Set Setting
https://img.shields.io/badge/HERO13 Black-4cbee0 https://img.shields.io/badge/HERO12 Black-566bb7 https://img.shields.io/badge/HERO11 Black Mini-e0952a https://img.shields.io/badge/HERO11 Black-7a4ea5 https://img.shields.io/badge/HERO10 Black-9bb859 https://img.shields.io/badge/HERO9 Black-a93b85
Type
:
TLV command

ID
:
SettingID
Set an individual Setting ID to a given value.

The superset of per-setting values can be found in the Setting ID documentation. Alternatively, the currently supported setting values can be dynamically queried via Get Setting Capabilities

Request
UUID
:
Setting

Parameters

value_length (uint8) - length of value
value (variable length and type) - value to set, of length value_length
Response
UUID
:
